## 🎮 Introduction
Play the classic game of Rock-Paper-Scissors against the computer right in your terminal!

---

## 📝 How to Play

### 1️⃣ Run the Program
Execute the following command in your terminal to start the game:
python rock_paper_scissors.py

2️⃣ Game Setup
Follow the on-screen instructions to choose the number of rounds you want to play.

3️⃣ Make Your Choice
After each countdown, input your choice:

Computer Wins: Type c
You Win: Type u
Tie: Type t

4️⃣ Game Outcome
The computer will randomly choose its action (rock, paper, or scissors) after each countdown.

5️⃣ Winner Announcement
After the specified number of rounds, the program will announce the winner based on the total score.

🎯 About the Game
Rock-Paper-Scissors is a classic hand game usually played between two people. Each player simultaneously forms one of three shapes with their hand:

Rock (a fist)
Paper (an open hand)
Scissors (a fist with the index and middle fingers extended, forming a V)

📜 Game Rules
Rock crushes scissors
Scissors cuts paper
Paper covers rock
This program mimics the real-life game, where players show their hand gesture at the same time to determine the winner of each round.

⚙️ Program Flow
1️⃣ Game Start
The program begins by asking you how many rounds you want to play.

2️⃣ Computer's Choice
It then initiates a countdown for the computer's choice (rock, paper, or scissors).

3️⃣ Player Input
You input your choice (c, u, or t) indicating who you think won the round.

4️⃣ Game Repeats
Steps 2-3 are repeated for the specified number of rounds.

5️⃣ Final Result
After all rounds are completed, the program announces the winner based on the total score.
